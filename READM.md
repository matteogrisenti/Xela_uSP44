#### Collect Data
To collect data you need to:
1. Launche the server
```bash 
cd ~/.local/bin
./xela_server
```

2. In another terminal launch the recorder:
```bash 
cd ~/.local/bin
./xela_log
```

3. The recoder are store bot as log and scv file in the ~ect/xela/CSV folder.
Copt the csv file inside this repository csv_records to use as beasilene for the different tools
